Other Projects

Aquarium Part 1

This Aquarium project was a part of an Object-Oriented Software Design course taken in Fall of 2018. It uses inheritance and parent child relationships to allow for the Carp to "eat" the other fish when being dragged over them. You are able to add and remove fish from the aquarium using the top menu. Position the fish by clikcing and dragging.

Aquarium Part 2

The second part of the Aquarium project focuses on using an update function to allow the fish to move as well as using xml to allow you to load and save aquariums. The "eat" function from part 1 has been removed, however you can still move the fish.
